Task Overview

Create a Model that can predict if a twitter tweet is sarcastic or not.


Referencing F1 Score of Sarcasm Class

  • Minimum:    0.275
  • Threshold:   0.4
  • Peak:           0.605

Plan of Action


  1. Develop upon a baseline reference model
  2. Achieve minimum requirements, and progress from there.
  3. Combine source code with winning code research paper.

Code Breakdown


Preprocessing

  • Gather data, replace emojis, urls, and usernames for generics
  • Create additional non-sarcastic statements
  • Combine and Divide Test and Validation Set, load onto Data Loaders

Training

  • Trains selected model off of preprocessed dataset
  • Runs code off of validation sets, and reports scores
  • Saves the best scores

Predictions

  • Uses the previously acquired best model to then run on source

Models

  • Run time varies significantly based on model, and epochs
